input:입력 요소

text type 입력 폼 / 문자열을 입력받는다.

password type 문자열을 입력받는다. 하지만 그 내용은 알 수 없다.

radio type 여러 보기중에 하나만 선택할 때 사용한다

check box 항목으로 선택 할지 말지 결정할 때 사용한다

'color' 항목으로 색을 지정할 수 있는 컨트롤. 브라우저가 지원하는 경우, 활성화 시 색상 선택기를 열어줍니다

'date 항목' 으로 날짜(연월일, 시간없음)을 지정할 수 있는 컨트롤, 브라우저가 지원하는 경우, 활성화시 날짜를 선택할 수 있는 달력 등을 열어줍니다 .

'email 항목 '이메일 주소를 편집할 수 있는 필드 . 텍스트 입력 필드처럼 보이지만 유효성 검증 매개변수가 존재하며, 브라우저와 장치가 동적 키보를 지원하는 경우 이메일에 적합한 키보드를 보여줍니다.
